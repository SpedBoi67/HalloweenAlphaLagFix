# Secret Neighbor Halloween Alpha Lag Fix
This repository contains files that lower graphics settings in the halloween alpha, reducing lag and increasing FPS.

⚠️ WARNING: THIS PATCH ONLY WORKS FOR V0.8.12 ⚠️

Instructions: Drag the downloaded assembly files to (WHATEVER NAME THE GAME FOLDER IS)/Neighbour_Data/Managed
